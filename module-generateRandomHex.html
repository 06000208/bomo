generateRandomHex

Small utility for generating random hexadecimal strings

Source: util/generateRandomHex.js, line 1

Methods

(inner) generateRandomHex(mapopt, nullable, bytesopt, allowedRecursionsopt):(nullable) string

Generates a string of random hex bytes for use as an id. Due to its tiny size by default (2 bytes to the power of 16 = 65536 possible ids, 4 characters per id), this is easily prone to collisions and may perform recursion checking against a Map's keys to avoid them

Parameters
map:Map = null (optional)
(nullable)

A map for optional collision checking

bytes:number = 2 (optional)

The number of bytes generated, by default 2, which generates ids 4 characters long

allowedRecursions:number = 256 (optional)

The number of permitted recursions, by default 256

Returns
Type
:string
  • the ID generated, null if it couldn't create a unique ID
